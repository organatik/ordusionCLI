Engineering Change Control Management

Engineering Change Control Management is an application that provides production managers with a simple and integrated interface for creating and tracking changes for products in a manufacturing environment. With a Web Portal primary interface this application is totally implemented in ABAP.

Quick facts about ECM:
  • A process to search, create, approve and track change request
  • Provide visibility to product under change control
  • Identify documents, BOM, Drawings, Cost Information, etc affected by change
  • Capability to adopt industry standard method (CMII)
  • Provide consistent change management process across the enterprise
  • A web based application accessible thru portal
  • WD4A based architecture
  • Utilizes SAP PLM core components
Benefits:
  • Product Data
  • Readily available when required
  • Single Source
  • Search and Re-use
Power of Change Control it provides:
  • Track changes against a product
  • Items affected by change
  • Revisions resulting in change
  • Engineering related Vs manufacturing related
  • Consistent approval process
  • Paths for fast track and routine changes
